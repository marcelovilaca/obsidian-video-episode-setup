# Guest Outreach & Booking Workflow

Use this note to manage guest acquisition from first contact to post-episode follow-up.

## Funnel
1. Prospect
2. Research
3. Outreach
4. Follow-up
5. Booked
6. Prep
7. Recorded
8. Published
9. Follow-up

## Step 1 — Prospecting
- [ ] Add the guest to the database
- [ ] Define why this guest fits the channel
- [ ] Check language / timezone fit
- [ ] Identify the best episode angle

## Step 2 — Research
- [ ] Review recent interviews, posts, and positions
- [ ] Identify current topic hooks
- [ ] Note thumbnail / title opportunities
- [ ] Note any risk factors or sensitive points

## Step 3 — Outreach
- [ ] Draft first message
- [ ] Draft follow-up message
- [ ] Include value proposition
- [ ] Include format, language, and estimated duration
- [ ] Include scheduling link or next step

## Step 4 — Follow-up
- [ ] Wait period noted
- [ ] Follow-up sent
- [ ] Response recorded
- [ ] Second follow-up if needed

## Step 5 — Booking
- [ ] Date confirmed
- [ ] Timezone confirmed
- [ ] Platform confirmed
- [ ] Recording / livestream format confirmed
- [ ] Guest note created or updated

## Step 6 — Prep
- [ ] Research dossier ready
- [ ] Questions drafted
- [ ] Thumbnail angle drafted
- [ ] Intro script drafted
- [ ] Tech / audio notes ready

## Step 7 — Post-episode
- [ ] Thank-you message sent
- [ ] Guest links shared
- [ ] Clips identified
- [ ] Future collaboration idea logged

## Best practice
Keep guests moving through the funnel instead of leaving them in a vague “maybe” state.
