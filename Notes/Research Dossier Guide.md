# Research Dossier Guide

Use the research dossier whenever you are preparing a new video, livestream, or guest episode.

## When to use it
- Before scripting
- Before booking a guest
- Before recording a debate or analysis
- Before creating the thumbnail and description

## What it should contain
- a one-paragraph summary of the topic
- source list
- key facts
- contradictions or uncertainties
- story angles
- question ideas
- clip candidates
- packaging cues
- recommendation for the best format

## Recommended workflow
1. Create a new dossier from the template.
2. Add sources and key facts.
3. Note contradictions or weak claims.
4. Decide the angle.
5. Use the dossier to write the script and package the video.
6. Link it back to the project note.

## Best practice
Never build the script before the dossier exists if the topic is complex or news-driven.
