# New Video Project Click Launcher

This is the safest click-based launcher with the current setup.

## One-click start
- [Open the launcher](file:///C:/Users/msoares/Downloads/NewVideoProjectLauncher.bat)

## What happens
1. You click the launcher file.
2. It asks for the theme, guest choice, and date.
3. It creates the full project folder automatically.
4. It writes the AI prompt note into the folder.
5. It opens the finished folder in Explorer.

## When to use this
Use this when you want a simple click from Obsidian without installing a shell-command plugin.
