## Usage
- Each guest should also have an individual note using the Podcast Guest Template.
- Update the database after every booking, recording, or follow-up.
- Use this note to decide who to invite next.
- New guest notes can now be created from the one-click project launcher when Guest = new.
- The launcher guest dropdown is populated from all notes tagged [[Podcast guests]], so past guests should appear automatically.
- Guest images are stored in Attachments/Facial References and linked in the database and guest notes.
