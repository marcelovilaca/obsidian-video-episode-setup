- Use the ribbon buttons for one-click video, livestream, or guest launches.
- Use the command palette if you want to choose the launch from text instead of clicking an icon.

## What to use when

### Option 1 — Fastest and easiest
Use the Obsidian plugin if it is installed.
- It asks for the theme, guest choice, and date
- If Guest = new, it also asks for Name, Country, Profession, and Language
- It creates the folder automatically
- It writes the AI prompt note and copies the prompt when possible
- It opens the folder when finished
- It also adds the guest to the guest database and category note
- It also links the facial reference image when one exists

### Option 2 — Manual but still safe
Use the Windows launcher file.
- No plugin needed
- Still one click from Obsidian
- Writes the AI prompt note
- Opens the folder when finished

## Recommended daily flow
1. Open this note.
2. Click the launcher file.
3. Fill in the theme, guest choice, and date.
4. Open the created project folder.
5. Fill the project hub note.
6. Move through idea → research → script → packaging → QA → publish → repurpose → review → archive.
