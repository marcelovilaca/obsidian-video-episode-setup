# Video Launch QA Checklist

Use this note before every publish.

## Title / thumbnail / hook
- [ ] Title is clear and clickable
- [ ] Thumbnail is readable on mobile
- [ ] Hook matches the title
- [ ] Thumbnail and title tell one coherent story
- [ ] First 15 seconds are strong

## Metadata
- [ ] Description written
- [ ] Tags / hashtags added if used
- [ ] Chapters added
- [ ] Pinned comment drafted
- [ ] Playlist selected
- [ ] End screen added
- [ ] Cards added

## Technical checks
- [ ] Video export verified
- [ ] Audio level checked
- [ ] Captions checked
- [ ] Aspect ratio checked
- [ ] Link check on mobile
- [ ] Link check on desktop
- [ ] Thumbnail looks good in small size

## Publishing
- [ ] Upload complete
- [ ] Privacy setting correct
- [ ] Schedule correct
- [ ] Final preview checked
- [ ] Publish / schedule confirmed

## Post-launch
- [ ] Check 2h performance
- [ ] Check 24h performance
- [ ] Reply to comments
- [ ] Log findings in project note
- [ ] Move card to Published
