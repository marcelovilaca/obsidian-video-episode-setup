# YouTube Creator OS

This note is the hub for the channel workflow. It is based on the current structure of the vault and tailored to the channel style of Imprensa de Destruição em Massa: geopolitical analysis, urgent news, interviews, livestreams, shorts, and heavy repurposing.

## Channel strategy snapshot

- Core lane: geopolitics, war, energy, elections, and global power shifts
- Tone: analytical, provocative, urgent, journalistic, sometimes sarcastic
- Primary language: Portuguese
- Best-performing format pattern: long-form analysis + livestreams + shorts cut from the main content
- Strong repeat topics: Iran, USA, Ukraine, Europe, China, Brazil, sanctions, energy, global order
- Guest-friendly lane: military analysts, journalists, geopolitical commentators, regional experts

## Operating rules

1. One note per content piece.
2. Every content piece gets a title, hook, thumbnail, description, and repurposing plan.
3. Publish fast when the topic is hot.
4. Do not fragment a topic into too many notes.
5. After publishing, log metrics and decide the next adjacent topic.
6. Treat guests as reusable assets with prep notes.
7. Keep prompt notes separate from project notes.

## Recommended vault workflow

### 1. Idea capture
- Store raw ideas in a content idea note
- Add source links, trend reason, and urgency score
- Decide if the idea is a video, livestream, interview, or short

### 2. Research
- Collect sources and transcript notes
- Pull key claims and supporting evidence
- Build a question bank if the piece is an interview or livestream

### 3. Scripting
- Create hook, structure, talking points, and CTA
- Generate a clean script or a run-of-show
- Add timestamps for clip candidates

### 4. Packaging
- Create title options
- Create thumbnail prompt(s)
- Draft description and pinned comment
- Prepare hashtags/tags when needed

### 5. Publishing
- Upload, verify metadata, and schedule
- Add chapters, end screens, cards, and playlist placement
- Confirm mobile and desktop link checks

### 6. Repurposing
- Cut Shorts, Reels, TikToks, X posts, community posts, and newsletter copy
- Reuse the same core idea in adjacent formats
- Cross-link every derivative asset back to the main video

### 7. Analytics
- Review 2h, 24h, and 7d signals
- Track CTR, retention, comments, and shareability
- Log what worked and what to repeat

## Recommended daily routine

- Morning: check the Daily Digest and Agents To Do, then review the news/idea queue
- Before writing: open the content project note and a relevant prompt note
- Before recording: finalize hook, title, and thumbnail angle
- After publishing: generate repurposed assets immediately
- End of day: update analytics and mark next action

## Suggested improvements

- Turn the old TXT prompts into reusable Markdown prompts with clear inputs/outputs.
- Keep separate prompt notes for description, script, thumbnail, repurposing, and guests.
- Use a guest database so recurring guests do not need to be rebuilt each time.
- Add a content pillar field to every project.
- Create a simple status model: idea → research → script → edit → package → publish → repurpose → archive.
