# New Video Project Folder Template

> [!info] Use this template
> Duplicate the project folder template, rename it, then open the project hub note.
> This gives you a fast, repeatable start for any new video or livestream.

## 1) Project setup
- [ ] Duplicate `Projects/Video Projects/_TEMPLATE - New Video Project/`
- [ ] Rename the folder to `YYYY-MM-DD - topic-slug`
- [ ] Open `00 - Project Hub.md`
- [ ] Confirm the content type: video / livestream / hybrid
- [ ] Set the pillar, status, and publish date

## 5) Keep the project folder clean
- [ ] Save all project assets inside the folder
- [ ] Put source links in the Sources area
- [ ] Put exports in the Exports area
- [ ] Put clips in the Clips area
- [ ] Archive the folder only after publishing and repurposing are done

## 6) Daily operating rule
Use the folder as the single source of truth for that video.
Do not split the same project across multiple unrelated notes.
