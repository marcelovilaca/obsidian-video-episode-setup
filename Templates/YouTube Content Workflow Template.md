# 🎬 {{Video Title}} — Content Workflow Tracker

> [!info] How to use this note
> - Duplicate this template for every new video or livestream
> - Keep one note per piece of content
> - Check items off as you complete them
> - Add links, timestamps, and decisions as you go
> - Archive the note when the content has been published, repurposed, and reviewed

## 1) Content Brief
- [ ] Define the core idea and audience problem
- [ ] Write the hook in one sentence
- [ ] Decide the format: long-form video / livestream / hybrid
- [ ] List the main takeaways or beats
- [ ] Note the CTA for this content

## 2) Pre-Production
- [ ] Script or outline finalized
- [ ] B-roll, screen recordings, overlays, and graphics list created
- [ ] Equipment and streaming setup tested (mic, camera, lighting, OBS/Streamlabs)
- [ ] Recording location and backups ready
- [ ] Title ideas drafted (at least 3)
- [ ] Thumbnail concepts drafted (at least 3)
- [ ] Keywords, tags, and category planned
- [ ] Publish date / livestream date confirmed
- [ ] Calendar updated and reminder created

## 3) Production
- [ ] Record the main content or go live
- [ ] Save raw files to: `/content/{{date}}-{{content_slug}}/`
- [ ] Log timestamps for strong moments, quotes, and cutdowns
- [ ] Save chat log / stream notes if livestream
- [ ] Backup footage locally and to cloud storage
- [ ] Record any issues to fix before editing

## 4) Post-Production
- [ ] Edit the main version
- [ ] Trim dead time and improve pacing
- [ ] Polish audio and color
- [ ] Add captions, chapters, end screens, and cards
- [ ] Export the master file
- [ ] Export repurposing versions (9:16, 1:1, text-safe crops)
- [ ] Finalize the thumbnail
- [ ] Draft description, pinned comment, and links
- [ ] Prepare chapter timestamps and source references

## 5) Publish / Go Live
- [ ] Upload in unlisted mode first for QA
- [ ] Verify title, thumbnail, description, tags, and chapters
- [ ] Check captions, end screens, cards, and playlist placement
- [ ] Set to Public or Scheduled
- [ ] Post teaser in Community / Discord / email / socials
- [ ] Confirm all links work on mobile and desktop

## 6) Repurposing Pipeline
> [!tip] Recommended order
> Repurpose the main piece immediately after publishing while momentum is high.

- [ ] Day 0: Long-form video or livestream goes live
- [ ] Day 0-1: Create 3-5 short clips for Shorts / TikTok / Reels
- [ ] Day 1-2: Create quote graphics or carousel slides
- [ ] Day 2-3: Write a thread or post summary for X / LinkedIn
- [ ] Day 3-5: Turn key points into a newsletter, blog post, or community post
- [ ] Day 5-7: Use comments or audience questions for follow-up content
- [ ] Add links from every derivative piece back to the main video
